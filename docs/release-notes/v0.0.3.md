# v0.0.3

## 릴리즈 개요

- 버전: `v0.0.3`
- 날짜: 2026-02-22
- 비교 기준: `v0.0.2`
- 성격: 다국어 UI, 모달 구조, Windows 패키징 CI 정비

## 주요 기능

- i18n hook과 다국어 번역 키를 확장했습니다.
- 상세/설정 모달의 기능 흐름을 정리하고 팝업 구조를 개선했습니다.
- preload와 renderer 사이의 IPC 브리지와 타입을 확장했습니다.

## UI/UX 개선

- 상세 화면과 설정 화면의 모달 UX를 통합된 패턴으로 정리했습니다.
- README에 npm 기반 실행 지침을 보강했습니다.

## 배포 및 CI

- Electron과 electron-builder 의존성 위치를 조정해 패키징 환경을 정리했습니다.
- Windows 패키징과 CI 워크플로를 추가했습니다.
- 네이티브 의존성 설치, `package-lock.json`, npm CI, Node.js 버전 설정을 정리했습니다.
- CI 패키징 단계에서 앱 빌드가 먼저 실행되도록 조정했습니다.
