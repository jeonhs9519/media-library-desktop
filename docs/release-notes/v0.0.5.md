# v0.0.5

## 릴리즈 개요

- 버전: `v0.0.5`
- 날짜: 2026-03-07
- 비교 기준: `v0.0.4`
- 성격: HDT 가져오기, 필터, 폴더 경로 일괄 변경, PDF 뷰어 개선

## 주요 기능

- HDT import UX를 개선하고 renderer API 타입을 정리했습니다.
- 실시간 카운트 기반 폴더 경로 일괄 변경 기능을 추가했습니다.
- 파일 누락 항목 검색 필터를 추가했습니다.
- PDF 뷰어에 화면 맞춤과 2페이지 보기 모드를 추가했습니다.

## UI/UX 개선

- 라이브러리 필터와 배지 UI를 개선했습니다.
- 열람 상태 필터를 고도화했습니다.
- 상세보기의 출처 URL을 시스템 기본 브라우저로 열도록 변경했습니다.

## 배포 및 CI

- 앱 아이콘 생성 파이프라인을 추가하고 Windows 패키징 아이콘을 적용했습니다.
- CI 패키징 실패를 줄이기 위해 prune 제거와 `publish never` 설정을 적용했습니다.
- CI 아티팩트 업로드 범위를 ZIP 파일로 제한했습니다.
