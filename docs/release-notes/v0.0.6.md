## 릴리즈 개요

- 버전: `v0.0.6`
- 날짜: 2026-03-28
- 비교 기준: `v0.0.5`
- 성격: 썸네일/메타데이터 자동 보강, 뷰어 UX 확장, 포터블 데이터 경로 정리

## 주요 기능

- 썸네일 표시를 개선하고 메타데이터 백그라운드 자동 채우기 범위를 확장했습니다.
- 동영상 뷰어에 커스텀 컨트롤과 컨텍스트 메뉴 UX를 추가했습니다.
- 만화책 뷰어에 오버레이, 페이지 모드, 단축키, 컨텍스트 메뉴 개선을 적용했습니다.

## UI/UX 개선

- 상세보기 팝업의 메타데이터 표시를 개선했습니다.
- 라이브러리 검색 조건을 유지하고 파일 상태 필터를 확장했습니다.
- 경로 변경 실패 시 조치 대상 파일 정보를 Modal에 표시하도록 개선했습니다.

## 구조 및 성능

- CBZ/PDF 뷰어 UI 공통화와 상태 관리 분리를 진행했습니다.
- renderer API wrapper를 적용했습니다.
- 포터블 데이터 경로 처리를 정리했습니다.
