## 릴리즈 개요

- 버전: `v0.0.1`
- 날짜: 2026-02-21
- 비교 기준: 첫 릴리즈
- 성격: 초기 Electron 데스크톱 앱과 태그 기반 GitHub Release 자동화 구성

## 주요 기능

- Electron + React 기반 개인용 미디어 라이브러리 앱의 초기 구조를 구성했습니다.
- 로컬 파일을 등록하고 라이브러리 목록에서 관리하는 기본 화면을 만들었습니다.
- PDF, CBZ, 비디오 파일을 다루기 위한 기본 뷰어 흐름을 추가했습니다.
- SQLite 기반 로컬 데이터 저장 구조와 앱 실행 흐름을 연결했습니다.

## UI/UX 개선

- 라이브러리 목록, 상세 화면, 설정 화면으로 이어지는 기본 데스크톱 앱 화면 흐름을 구성했습니다.
- README에 설치와 실행 방법을 추가했습니다.

## 배포 및 CI

- Windows 환경에서 Electron 앱 실행과 DB 마이그레이션 흐름을 안정화했습니다.
