# Backlog

Last updated: 2026-04-29

아직 착수하지 않았거나, 범위 정리가 더 필요한 항목을 모아 둡니다.

## 태그 필터링

- 라이브러리 상단에 전체 태그 목록 표시
- `{태그명} ({건수})` 형식 사용
- 건수 내림차순 정렬
- 태그 목록 접기/펼치기
- 태그 선택 시 AND 조건 필터링
- 태그 전용 초기화 버튼
- `태그 미지정` 개념 도입 여부 검토
- 다국어 키 정리

## 프로필

- 최초 실행 시 프로필 생성 또는 선택
- 등록 프로필이 없는데 데이터가 있으면 `GUEST` 일괄 이관 정책 검토
- 프로필명 중복 방지
- 라이브러리와 태그를 프로필 범위로 관리
- 설정 팝업에서 전환, 이름 변경, 삭제 지원

## 플레이리스트

- 라이브러리 목록 좌우 중 위치 선택 가능
- 목록/상세 팝업에서 플레이리스트 추가
- 드래그 추가 또는 컨텍스트 메뉴 추가
- `book`, `comic`, `video`만 허용
- `other` 타입 차단 UX 정리
- 하단 액션: `뷰어 열기`, `초기화`
- 완료 시 다음 항목 자동 실행

## 접근성 및 UX

- 라이브러리 카드 `Tab` 포커스 이동
- `Enter`로 상세 팝업 열기
- 상세 팝업 focus trap 정리
- 라이브러리 카드 우클릭 컨텍스트 메뉴
- `열기`, `뷰어 열기`, `외부에서 열기`, `출처 URL로 이동`, `파일 위치 열기`
- 뷰어 공통 단축키: `ESC`, `Backspace`

## 태그 관리 개선

- 저장/삭제 시 태그 사용률 재계산
- 미사용 태그 자동 정리
- `태그 미지정` 예약어 제한

## 문서/구조 개선

- `items` IPC 책임 분리
- `LibraryPage.tsx` 컴포넌트/훅 분리
- 테스트 공백이 큰 흐름부터 보호
