# Backlog

Last updated: 2026-05-01

아직 착수하지 않았거나, 범위 정리가 더 필요한 항목을 모아 둡니다.

## 프로필

- 최초 실행 시 프로필 생성 또는 선택
- 등록 프로필이 없는데 데이터가 있으면 `GUEST` 일괄 이관 정책 검토
- 프로필명 중복 방지
- 라이브러리와 태그를 프로필 범위로 관리
- 설정 팝업에서 전환, 이름 변경, 삭제 지원

## 플레이리스트

- 라이브러리 목록 좌/우 표시를 지원하고 기본값은 우측
- 설정 팝업의 `표시 설정`에서 좌/우 위치 선택
- 최초 실행 시 `width: 32px; height: 100%;` 형태의 접힌 상태로 표시
- caret으로 접기/펼치기 토글
- 접힌 상태에서 리스트에 항목 추가 시 자동 펼침
- 접힘/펼침 상태 전환 시 라이브러리 목록 방향키 이동 계산 갱신
- 라이브러리 목록 썸네일 우클릭 컨텍스트 메뉴에서 `리스트에 추가`
- 라이브러리 목록에서 플레이리스트로 drag and drop 추가
- 상세보기 팝업에서 `리스트에 추가` 버튼 제공
- 상세보기 팝업의 `리스트에 추가` 버튼은 `뷰어 열기`와 `외부에서 열기` 사이에 배치
- `book`, `comic`, `video`만 허용
- `other` 타입은 컨텍스트 메뉴/상세보기 버튼 비활성화
- `other` 타입 drag and drop 시 `cursor: disabled`
- 하단 액션: `뷰어 열기`, `초기화`
- 완료 시 다음 항목 자동 실행

## 접근성 및 UX

- 핵심 사용자 흐름 테스트 보강 시 상세 삭제 확인 Modal과 뷰어 썸네일 toast 동작을 함께 검증

## 문서/구조 개선

- 테스트 공백이 큰 흐름부터 보호
