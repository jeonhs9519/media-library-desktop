# Backlog

Last updated: 2026-05-02

아직 착수하지 않았거나, 범위 정리가 더 필요한 항목을 모아 둡니다.

## 프로필

- 최초 실행 시 `프로필 생성` Modal 제공
- 최초 실행 시 `GUEST` 시작 또는 신규 프로필 생성 선택
- 기존 데이터가 있으면 최초 실행에서 선택한 프로필로 이관
- 프로필명 중복 방지
- 리뷰, 진행률, 플레이리스트, 설정을 포함한 모든 내용을 프로필별로 분리
- 설정 팝업에서 전환, 이름 변경, 삭제 지원
- 프로필 삭제 시 데이터가 있으면 `GUEST`로 이관한 뒤 삭제
- 라이브러리 목록 썸네일 컨텍스트 메뉴에서 `다른 프로필로 이관`
- 상세보기 팝업의 `리뷰`와 `파일` 사이에 `다른 프로필로 이관` 영역 제공
- 프로필 이관 시 파일 정보, 진행률, 리뷰를 함께 이관

## 접근성 및 UX

- 핵심 사용자 흐름 테스트 보강 시 상세 삭제 확인 Modal, 뷰어 썸네일 toast, 플레이리스트 회귀 동작을 함께 검증

## 문서/구조 개선

- 테스트 공백이 큰 흐름부터 보호
