# Backlog

Last updated: 2026-04-30

아직 착수하지 않았거나, 범위 정리가 더 필요한 항목을 모아 둡니다.

## 프로필

- 최초 실행 시 프로필 생성 또는 선택
- 등록 프로필이 없는데 데이터가 있으면 `GUEST` 일괄 이관 정책 검토
- 프로필명 중복 방지
- 라이브러리와 태그를 프로필 범위로 관리
- 설정 팝업에서 전환, 이름 변경, 삭제 지원

## 플레이리스트

- 라이브러리 목록 좌우 중 위치 선택 가능
- 목록/상세 팝업에서 플레이리스트 추가
- 드래그 추가 또는 컨텍스트 메뉴 추가
- `book`, `comic`, `video`만 허용
- `other` 타입 차단 UX 정리
- 하단 액션: `뷰어 열기`, `초기화`
- 완료 시 다음 항목 자동 실행

## 접근성 및 UX

- 뷰어 공통 단축키: `ESC`, `Backspace`
- 카드 컨텍스트 메뉴에 추가 관리 액션을 넣을지 검토
- 라이브러리 목록 roving focus, 페이지네이션, focus return 동작을 실제 사용 환경에서 장기 사용하며 추가 점검
- 검색 툴바 아이콘 버튼의 의미가 충분히 전달되는지 다국어 `title`/`aria-label` 문구 점검

## 문서/구조 개선

- 테스트 공백이 큰 흐름부터 보호
