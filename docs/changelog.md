# Changelog

## 2026-05-05

- 앱 시작 시 과거 userData 위치의 `media-library.db`를 자동 복사해 사용하는 흐름을 제거
- 설정 팝업에 `과거 데이터 불러오기` 항목을 추가하고 DB 파일 선택과 `불러오기` 버튼을 연결
- 기존 `media-library.db` 미리보기 Modal을 추가해 원본 항목 수, 가져오기 가능 항목 수, 중복/제외 항목 수, 태그/리뷰/플레이리스트 수를 확인할 수 있게 함
- 기존 DB 가져오기 IPC를 추가해 중복되지 않은 `items`와 연결 가능한 `tags`, `itemTags`, `reviews`, `settings`, `playlists`, `playlistItems`를 현재 DB로 이관
- 기존 DB 가져오기 후 라이브러리 목록을 다시 불러오도록 연결
- 과거 DB 가져오기 파일 선택 UI를 설정 팝업의 폴더 경로 일괄 변경 입력 행과 같은 형태로 조정
- 과거 DB 가져오기 미리보기 Modal을 `설정`, `태그`, `파일 정보` 아코디언 구조로 변경하고 파일 정보에 진행률, 썸네일, 태그, 리뷰 상태를 표시
- 과거 DB 가져오기 미리보기 Modal 폭을 720px로 고정하고, 중복/제외 항목 opacity를 낮춰 가져오지 않는 항목을 더 명확히 표시
- 과거 DB 가져오기 미리보기 Modal의 태그 아코디언 최상단에 미사용 태그 자동 삭제 안내 문구를 추가
- 라이브러리 목록 상단 툴바의 `HDT 가져오기` 버튼을 제거하고, 설정 팝업의 `과거 데이터 불러오기` 아래에 `HDT 가져오기` 항목을 추가
- 설정 팝업의 `HDT 가져오기`도 readonly 입력, 숨김 파일 입력, `불러오기` 버튼 방식으로 맞춤
- 설정 팝업의 `과거 데이터 불러오기` 항목을 폴더 경로 일괄 변경 아래로 이동
- 설정 팝업의 파일 선택형 항목 버튼, 일괄 변경 버튼, footer 버튼 폭과 행 정렬을 조정
- `HDT 일괄 가져오기` Modal이 설정 팝업 위에 표시되도록 공용 Modal의 선택적 `zIndex`를 추가
- 아코디언 caret 표시를 SVG 아이콘으로 교체하고 `CaretBottomIcon`을 추가
- `npm.cmd run build`, `npm.cmd test -- --run` 통과 확인
