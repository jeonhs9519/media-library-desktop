### 1. 플레이리스트 구현

- 라이브러리 목록 좌/우 모두 표시 가능하게 구현합니다.
- 기본값은 우측입니다.
- 설정 팝업의 `표시 설정` 항목에서 좌/우 위치를 선택할 수 있게 합니다.
- 최초 실행 시 라이브러리 목록 기준 `width: 32px; height: 100%;` 형태의 접힌 상태로 표시합니다.
- 접힘/펼침은 caret 버튼으로 토글합니다.
- 접힌 상태에서는 `<` 모양으로 펼칠 수 있게 표시합니다.
- 펼쳐진 상태에서는 `>` 모양으로 접을 수 있게 표시합니다.
- 접힌 상태에서 리스트에 항목이 추가되면 자동으로 펼칩니다.
- 창 크기 변경뿐 아니라 접힘/펼침 상태 전환 시에도 라이브러리 목록 방향키 이동 계산을 갱신합니다.
- 라이브러리 목록 썸네일 우클릭 컨텍스트 메뉴에 `리스트에 추가` 메뉴를 추가합니다.
- 라이브러리 목록에서 플레이리스트로 drag and drop 추가를 지원합니다.
- 상세보기 팝업에 `리스트에 추가` 버튼을 추가합니다.
- 상세보기 팝업의 `리스트에 추가` 버튼 위치는 `뷰어 열기`와 `외부에서 열기` 사이입니다.
- 지원 대상은 `book`, `comic`, `video`입니다.
- `other` 타입은 목록 컨텍스트 메뉴의 `리스트에 추가`를 비활성화합니다.
- `other` 타입을 플레이리스트로 drag and drop할 때 `cursor: disabled`를 적용합니다.
- `other` 타입은 상세보기 팝업의 `리스트에 추가` 버튼을 비활성화합니다.
- 하단 액션은 `뷰어 열기`, `초기화`를 제공합니다.
- 완료 시 다음 항목 자동 실행 조건을 구현 전에 다시 좁혀서 정리합니다.
