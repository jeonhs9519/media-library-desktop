## 바로 시작하기 좋은 작업

### 1. 프로필 구현

- 최초 실행 시 프로필이 없으면 `프로필 생성` Modal을 표시합니다.
- `프로필 생성` Modal에서는 `GUEST` 시작 또는 신규 프로필 생성을 선택할 수 있게 합니다.
- 기존 데이터가 있으면 선택된 프로필로 이관된다는 안내를 표시합니다.
- 리뷰, 진행률, 플레이리스트, 설정을 포함한 모든 내용을 프로필별로 분리합니다.
- 설정 팝업 본문 최상단에 `프로필 관리` 항목을 추가합니다.
- 프로필 관리 항목에서는 현재 프로필 이름 변경, 프로필 전환, 프로필 삭제를 지원합니다.
- 프로필 삭제는 확인 Modal을 거치며, 데이터가 있으면 `GUEST` 프로필로 이관한 뒤 삭제합니다.
- 프로필 전환은 `프로필 선택` Modal로 처리하며, `GUEST`, 등록된 프로필, 신규 프로필 생성 입력을 제공합니다.
- 라이브러리 목록 썸네일 컨텍스트 메뉴에 `다른 프로필로 이관` 상위 메뉴를 추가합니다.
- 파일 상세보기 팝업의 `리뷰`와 `파일` 사이에 `다른 프로필로 이관` 영역을 추가합니다.
- 프로필 이관 시 파일 정보, 진행률, 리뷰를 지정한 프로필로 함께 이관합니다.
