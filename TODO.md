# 추가/개선 예정 항목

## 추가
- 태그를 이용한 필터링 기능 추가
    - 최초 실행 시, 라이브러리 상단에 전체 태그 표시
        - `N개 항목` 보다 위에 배치
        - `{태그명} ({건수})` 형태로 표시
        - 건수가 많은 순으로 내림차순 정렬
    - 태그 목록을 접고 펼 수 있도록 구현
        - 최초 실행 시, 항상 `펼침` 상태로 처리
        - 검색 결과가 2페이지 이상인 경우, 2페이지 이후로 이동 시 태그 목록 `접힘` 처리
    - 태그 목록에서 태그 선택 시, `AND` 조건으로 즉각 필터링 수행
        - 선택된 태그들을 모두 포함하는 파일로 라이브러리 필터링
        - 선택된 태그들을 모두 포함하는 파일들에 대해, 각각이 추가로 가지고있는 다른 태그들로 태그 목록 필터링
    - 태그에 대해 따로 초기화 버튼 추가
    - `태그 미지정`이라는 이름의 태그 추가
        - 다국어 지원
- 프로필 추가
    - 최초 실행 시 프로필을 추가/선택할 수 있도록 처리
        - 등록된 프로필이 없는데 파일 정보 등이 존재하는 경우, `GUEST` 명의로 일괄 이관
        - 등록 시 명칭 중복 검사 수행, 동일한 명칭으로 등록할 수 없게 처리
    - 라이브러리 목록과 태그에 대해 프로필 적용
    - 설정 팝업에 프로필 관련 버튼 추가
        - 프로필 전환: 최초 실행 시와 동일한 상태로 변경
        - 프로필명 변경:
            - 현재 프로필의 명칭을 변경
            - 변경 시 명칭 중복 검사 수행, 동일한 명칭으로 변경할 수 없게 처리
        - 프로필 삭제: 현재 프로필을 삭제하고, 해당 프로필에 등록된 라이브러리 및 태그 정보 일괄 삭제
            - 선택 시 경고 팝업 표시: (예: JavaScript)
                ```js
                '현재 프로필을 삭제합니다.<br/>' +
                '등록된 파일 정보와 태그가 모두 삭제되며, 복구할 수 없습니다.<br/>' +
                '- 등록된 파일 정보: {해당 프로필에 등록된 전체 파일 정보 수}건<br/>' +
                '- 등록된 태그 정보: {해당 프로필에 등록된 전체 태그 정보 수}건<br/>' +
                '정말로 삭제하시겠습니까?'
                ```
- 플레이리스트 기능 추가
    - 라이브러리 목록 좌측 또는 우측에 위치
        - 표시 위치는 설정에서 지정
    - 라이브러리 목록이나 상세보기 팝업에서 플레이리스트에 추가
        - 라이브러리 목록:
            - 썸네일을 드래그하여 직접 추가
            - 썸네일 우클릭 시 실행되는 컨텍스트 메뉴에 `리스트에 추가` 항목 추가
        - 상세보기 팝업: `뷰어 열기` 버튼과 `외부에서 열기` 버튼 사이에 `리스트에 추가` 버튼 추가
    - 단, 뷰어가 존재하는 도서(B), 만화(C), 동영상(V)만 추가 가능
    - 뷰어가 존재하지 않는 파일(F)인 경우, 플레이리스트에 추가를 시도할 시 다음과 같이 차단
        - 라이브러리 목록에서 썸네일을 드래그하려고 할 때, `cursor: disabled;` 처리
        - 라이브러리 목록에서 썸네일 우클릭 시, 컨텍스트 메뉴에서 `리스트에 추가` 항목 비활성화
        - 상세보기 팝업에 `리스트에 추가` 버튼 비활성화
    - 플레이리스트 하단에 `뷰어 열기` 버튼과 `초기화` 버튼 추가
        - `뷰어 열기`: 플레이리스트의 첫 번째 항목에 알맞는 뷰어로, 첫 번째 항목 실행
        - `초기화`: 플레이리스트 초기화
    - 플레이리스트에 추가된 항목에 대해, 진행률이 100%가 될 시 곧바로 다음 파일을 뷰어로 실행
        - 예: 8페이지 만화, 1분 동영상, 4페이지 만화로 구성된 경우
            - 8페이지 만화(첫번째 파일)의 8페이지 열람(진행률 100%) 후, 다음 페이지로 이동 시 1분 동영상(두번째 파일)을 동영상 뷰어로 실행
            - 1분 동영상(두번째 파일)을 1분간 시청(진행률 100%) 후, 곧바로 4페이지 만화(세번째 파일)를 만화책 뷰어로 실행
            - 4페이지 만화(세번째 파일)의 4페이지 열람(진행률 100%) 후, 동작 중단 (단일 파일 뷰어 실행 시와 동일)
    - 진행률은 각각의 파일에 대해 별도로 적용

## 개선
- 접근성 개선
    - 라이브러리 목록에서, `Tab` 키로 썸네일에 접근(= 포커스 이동)할 수 있도록 처리
    - 썸네일 선택 후 `Enter` 키로 상세 팝업 호출
    - 상세 팝업 호출 시, 제목으로 포커스를 이동시키고, 이후 `Tab` 키 입력 시 상세 팝업 내에서 포커스가 이동되도록 처리
- 라이브러리 목록에 컨텍스트 메뉴 적용
    - 썸네일 우클릭 시 실행
    - `열기`(상세 팝업 호출), `뷰어 열기`, `외부에서 열기`, `출처 URL로 이동`, `파일 위치 열기`로 구성
- 썸네일 내 아이콘 표시 방식 개선
    - 기존: 하단에 타입과 언어만 표시
    - 변경: 상단에 타입과 언어 표시, 우측 하단에 재생시간 또는 페이지 표시
        - 동영상: 시청시간/총시간
        - 도서, 만화: 열람 페이지/총 페이지
- 상세보기 팝업 중 메타데이터 표시 개선
    - 기존: 콘텐츠 타입, 언어 / 작성자, 시청 완료 / 진행률, 출처 URL
    - 변경: 콘텐츠 타입, 언어 / 작성자 / 진행률, 시청완료 / 출처 URL
        - 진행률 표기 방식 개선
            - 기존: `{진행률}%`
            - 변경: `{시청분량}/{전체분량} ({진행률}%)`
- 태그 관리 방식 개선
    - 저장/삭제 시, 전체 태그에 대해 사용률 확인
    - 사용하지 않는 태그는 삭제 처리하고, 라이브러리 목록의 태그 목록에도 반영
    - `태그 미지정`이라는 이름으로는 태그를 추가할 수 없도록 제한
        - 다국어 지원
- 검색 조건 관리 방식 개선
    - 현재 지정된 전체 검색조건을 항시 보유하도록 개선
    - 뷰어 화면에 다녀 온 뒤에도 검색조건 유지
    - 검색조건 초기화 버튼 추가
- 뷰어 공통 단축키 추가
    - `ESC`, `Backspace`: 뷰어 종료
- 도서/만화책 뷰어에 단축키 추가
    - `1`: 한 장 보기 / 스크롤 보기 전환
    - `2`: 두 장 보기 좌/우 전환
    - `zxcvbnm,./`: 이전 페이지 보기
    - `space`: 다음 페이지 보기
    - 오른쪽 방향키:
        - 다음 페이지: 한 장 보기, 좌->우 보기
        - 이전 페이지: 우->좌 보기
    - 왼쪽 방향키:
        - 다음 페이지: 우->좌 보기
        - 이전 페이지: 한 장 보기, 좌->우 보기
- 동영상 뷰어 개선
    - 커스텀 UI 적용
        - 5초 전, 재생/일시정지, 5초 후, 프로그레스바, 볼륨, 전체화면, 옵션으로 구성
        - 프로그레스바에 대해, `mouseout` 시점으로 이동 처리
    - 단축키 추가
        - 방향키
            - 왼쪽 방향키: 5초 되감기
            - 오른쪽 방향키: 5초 빨리감기
            - 위쪽 방향키: 볼륨 5% 증가
            - 아래쪽 방향키: 볼륨 5% 감소
        - `M`: 볼륨을 0%로 지정
        - `F`: 전체 화면 토글
    - 이어서 보기 적용
        - 시청 완료 상태가 아니면서 마지막 시청 시간 정보가 있을 경우, 해당 시간부터 재생 처리
        - 시청 완료 상태인 경우 처음부터 재생
    - 동영상에 우클릭 시 컨텍스트 메뉴를 추가
        - `재생`, `일시정지`, `전체화면`, `반복 재생`(신규), `파일 위치 열기`로 구성
    - 반복 재생 기능 추가
        - 진행률이 100%가 되었을 경우, 즉시 처음으로 되돌려 재생
        - 시청완료는 체크하고, 진행률은 마지막 재생 시점 기준으로 지정
    - 볼륨 정보 저장
        - 마지막으로 재생한 동영상을 기준으로, 볼륨 정보를 설정으로 저장
        - 이후 동영상 뷰어 실행 시, 해당 볼륨 정보를 적용하여 재생
